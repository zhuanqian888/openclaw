# OpenClaw 记忆仓库

本仓库存储 OpenClaw 助手的所有记忆文件，用于在不同环境间同步状态。

## 记忆文件说明

| 文件 | 用途 |
|------|------|
| **SOUL.md** | 定义助手的核心人格、价值观和行为准则 |
| **USER.md** | 记录用户（杰哥）的信息、偏好和上下文 |
| **IDENTITY.md** | 助手的身份标识（名称、性格、头像等） |
| **MEMORY.md** | 长期记忆：重要事件、决策和经验总结 |
| **AGENTS.md** | Agent 相关的通用配置和约定 |
| **TOOLS.md** | 工具/技能配置（的具体相机、SSH、TTS 等） |
| **HEARTBEAT.md** | 心跳检查任务清单（空=跳过） |

## 同步规则

- 每次更新记忆文件后，**自动同步到本仓库**
- 主要通过 Telegram 指令触发同步
- 保持本地与 GitHub 仓库一致
